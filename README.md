Aurele Product Catalog
A web application built with Python and Flask for managing a roduct catalog.
Features:
 -view all products
 -add new products
 -remove products
 -save data permanently to a JSON file
built with:
 -Python
 -Flask
 -Html
 -CSS
How to run:
 -install flask: pip install flask
 -run: python app.py
 -open browser: http://127.0.0.1:5000
